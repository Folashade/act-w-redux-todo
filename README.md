# README.md

## Terminal Commands
--------------------
### Setup
npm init
npm install react
npm install react-dom
npm install babel-loader babel-core babel-preset-es2015 babel-preset-react --save-dev
### Running the App
npm install webpack -g
webpack -w
### Add Redux
npm install --save redux

### Useful
unstage files : git reset HEAD -- .
